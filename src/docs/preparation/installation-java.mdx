# Java Installation Guide

## Windows

1. Öffnen Sie die <ins> <a href="https://adoptium.net/?variant=openjdk11&jvmVariant=hotspot" target="_blank">Adoptium Website</a></ins>. <!--- open link in new tab -->
2. Gehen Sie sicher, dass Temurin 11 ausgewählt ist und drücken Sie auf **"Latest Release"**. Der Download sollte automatisch starten.
3. Öffnen sie den soeben heruntergeladenen Installer und drücken sie auf **"weiter"**, bzw. **"installieren"**. Sie können die Standardeinstellungen übernehmen und müssen nichts weiteres im Installer auswählen.
4. Nachdem die Installation abgeschlossen ist, drücken sie **"Fertig stellen"**.
5. Sie können überprüfen, ob Java korrekt installiert wurde, indem Sie im Terminal folgenden Befehl eingeben
 
 >java --version

   * in Windows können Sie ein Terminal öffnen, indem Sie in der Windowssuche (Win + S) **"cmd"** eingeben.
   * Die Ausgabe sollte in etwa wie folgt aussehen:
   > openjdk 11.0.14.1 2022-02-08
