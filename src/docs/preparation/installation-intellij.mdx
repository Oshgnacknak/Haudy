# IntelliJ installieren

1. Laden Sie auf der <ins> <a href="https://www.jetbrains.com/de-de/idea/download" target="_blank">JetBrains Website</a></ins> entweder die Community Edition oder die Ultimate Edition herunter. Sie können mit beiden Versionen die Hausübung bearbeiten.
   * Die Ultimate Edition bittet Ihnen einige weiter Features und Sie erhalten sie als Student kostenlos. Dafür müssen sie sich <ins> <a href=" https://www.jetbrains.com/de-de/community/education/#students" target="_blank">hier</a></ins> mit Ihrer studentischen E-Mail registrieren.
   * Weitere Information zur studentischen E-Mail finden sie auf der Seite des <ins> <a href="https://www.hrz.tu-darmstadt.de/services/it_services/linux_mail/index.de.jsp" target="_blank">HRZ</a></ins>.
2. Führen Sie den heruntergeladenen Installer aus. Sie können die Standardeinstellungen des Installers übernehmen, indem sie auf **"next"** bzw. **"install"** drücken.
   * Die Option **"create Desktop Shortcut"** erstellt für Sie einen Shortcut auf dem Desktop, mit welchem Sie IntelliJ direkt öffnen können. 
   * Die Option **"update Context Menu"** ermöglicht es Ihnen Projekte direkt im File Explorer mit einem Rechtklick auf den entsprechenden Folder zu öffnen.
3. Drücken Sie zum Schluss auf **"finish"**, um den Installationsprozess abzuschließen.
4. Wenn Sie IntelliJ das erste Mal starten und die Ultimate Edition heruntergeladen haben, müssen sie sich noch mit Ihrem JetBrains Account, welchen sie in Schritt 1 erstellt haben, anmelden, indem sie auf **"Log in to JetBrains Account"** drücken und dann Ihre Anmeldedaten im Browser eingeben. Drücken sie danach in IntelliJ auf **"activate"**, um die Lizenz zu aktivieren. Danach können Sie das Fenster schließen, indem Sie auf **Continue** drücken.
5. Nun können Sie die Hausübung mit IntelliJ bearbeiten. Befolgen Sie dafür die Schritte der Anleitungen im Abschnitt **Hausübung**.
